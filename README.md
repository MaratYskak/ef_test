# Rick and Morty App

![Demo App](rm.gif)

## Описание проекта

Это мобильное приложение на Flutter, разработанное для просмотра списка персонажей из популярного мультсериала "Рик и Морти" с использованием публичного API (https://rickandmortyapi.com/). Приложение позволяет пользователям добавлять персонажей в избранное, обеспечивая при этом оффлайн-доступ к данным и сохранение избранных персонажей в локальной базе данных SQLite. Приложение поддерживает переключение между светлой и темной темами, а также включает кастомные анимации при взаимодействии с избранными.

Архитектура проекта основана на принципах Clean Architecture (feature-first), с использованием следующих ключевых библиотек и подходов:

*   **Dio**: Для выполнения HTTP-запросов к API.
*   **GetIt**: Для управления зависимостями (Dependency Injection).
*   **GoRouter**: Для навигации по приложению.
*   **BLoC (Business Logic Component)**: Для управления состоянием приложения.
*   **SQLite**: Для локального хранения данных избранных персонажей.
*   **CachedNetworkImage**: Для эффективной загрузки и кеширования изображений.
*   **SharedPreferences**: Для сохранения настроек пользователя (например, выбор темы).

Приложение также реализует адаптивную верстку для корректного отображения на различных устройствах и пагинацию для подгрузки новых персонажей при скролле.

## Инструкция по сборке и запуску

Для сборки и запуска приложения выполните следующие шаги:

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/MaratYskak/ef_test.git
    cd rick_and_morty_app
    ```

2.  **Установите зависимости Flutter:**

    ```bash
    flutter pub get
    ```

3.  **Запустите приложение:**

    ```bash
    flutter run
    ```

    Приложение будет запущено на подключенном устройстве или эмуляторе.
